📊 EDA IPL Project – Exploratory Data Analysis of IPL Dataset (Python)

A Python-based Exploratory Data Analysis (EDA) project performed in Google Colab using IPL match data from multiple seasons.
The goal is to clean, analyze, and visualize patterns related to teams, venues, toss decisions, player performances, and match outcomes.

🚀 Project Overview

This project analyzes 15+ seasons of IPL data to uncover key insights such as:

Toss impact on match outcomes
Most successful teams
High-scoring venues
Top “Player of the Match” performers
Home-ground advantages
Batting and bowling patterns

The analysis includes data cleaning, feature exploration, visualizations, and summary conclusions.

📌 Key Insights
Final Conclusions from IPL EDA
Toss does not determine the winner — match wins are almost evenly split regardless of toss result.
Some venues are consistently high-scoring, influencing batting-first performance.
Home teams gain a slight advantage at their home grounds.
Teams like MI, CSK, and KKR are most consistent, securing the highest number of wins.
Star players such as Kohli, Dhoni, and Gayle dominate Player of the Match awards, showing long-term individual impact.
